# LinguoLand

> 在阅读中学英语 —— 一个以「大量真实阅读输入」为核心的英语学习产品。

LinguoLand 是一套围绕「读」展开的英语学习工具：你导入或选择想读的内容，遇到生词、长难句随手查、随手收藏，词汇与语法的提升在阅读主线下顺带发生，而不是被抽成枯燥的背词任务。

本仓库是 LinguoLand 的**公开门面**——用来收集用户反馈（Issues）、发布公开文档，以及开放一些可复用的资产（如 [skills](#skills)）。产品的应用代码不在这里。

## 产品矩阵

| 端 | 说明 |
| --- | --- |
| 浏览器扩展 | 在任意网页上即点即查、收藏生词与句子 |
| 移动端阅读器 | iOS / Android，沉浸式阅读 + 词汇语法辅助 |
| Web 阅读器 | 浏览器内阅读，与移动端阅读功能对齐 |
